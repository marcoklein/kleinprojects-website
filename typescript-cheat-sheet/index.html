Typescript Cheat Sheet

Decorators

Can be attached to

  • class
  • class method
  • class property
  • class method parameter

Benefits

  • logging, e.g. @deprecated

Class decorators

function decorator(options: { prop: string }) {
  // target is our class prototype
  return target => {
    // ...
  };
}

@decorator({
  prop: 'foo',
})
class MyClass {
  // ...
}
